# @react-lib

A react component library

## Installation

```bash
bun add @anitshrsth/react-lib
```

## Usage

```tsx
import '@react-lib/styles.css';
import { Button } from '@react-lib';

function App() {
  return <Button>Click me</Button>;
}
```
